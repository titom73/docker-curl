# cURL in docker

Alpine-based image with just curl

## Usage

```console
$ docker run --rm titom73/curl -fsSL https://www.google.com/
```

## Tags

* `titom73/curl:latest`: based on `alpine:latest`
* `titom73/curl:edge`: based on `alpine:edge`
